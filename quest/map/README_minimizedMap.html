This README assumes that you have already read the other main map README. #################################### Minimized Mode: The minized mode can be activated in the /server/Db_quest file by setting Quest.test.minimized = true; When minimized is on: -Monsters can't be attacked. Instead, they can be killed by right-clicking and selecting "Kill" instead. (To trigger deathFunc) -Challenge, highscore, requirements, chrono, etc... will do nothing. -There's only 1 map with everything in it. (s.teleport will do nothing.) -q.map and q.mapAddon will not work. (You need to use q.minimizedMap instead.) -Loop for maps will do nothing. q.minimizedMap acts as the load function of a regular map. In q.minimizedMap you can only use the following functions: ##### Create a simplified actor: m.minimized.actor('name',extra); Ex: s.minimized.actor('Dragon',{deathFunc:q.event.killDragon}); Create a simplified loot (aka chest): m.minimized.loot('name',viewedIf,open); Ex: s.minimized.loot('Jewel Chest',q.event.gotJewel,q.event.giveJewel); Create a simplified loot (aka chest): m.minimized.loot('name',viewedIf,on,off); Ex: s.minimized.loot('Door Switch',q.event.doorIsLocked,q.event.unlockDoor,q.event.lockDoor); ##### q.minimizedMap = function(){ //create these actors when creating/loading the map s.minimized.actor('Dragon',{deathFunc:q.event.killDragon}); s.minimized.loot('Jewel Chest',q.event.gotJewel,q.event.giveJewel); s.minimized.loot('Door Switch',q.event.doorIsLocked,q.event.unlockDoor,q.event.lockDoor); }
