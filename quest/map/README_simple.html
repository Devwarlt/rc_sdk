This README assumes that you have already read the other main map README. #################################### Minimized Mode: The minized mode can be activated in the /server/Db_quest file by setting Quest.test.minimized = true; When minimized is on: -Monsters can't be attacked. Instead, they can be killed by right-clicking and selecting "Kill" instead. (To trigger deathFunc) -Challenge, highscore, requirements, chrono, etc... will do nothing. -There's only 1 map with everything in it. (s.teleport will do nothing.) -q.map and q.mapAddon will not work. (You need to use q.minimizedMap instead.) -Loop for maps will do nothing. q.minimizedMap acts as the load function of a regular map. In q.minimizedMap you can only use the following functions: ##### Create a simplified actor: m.actor.simple('name',extra); Ex: m.actor('Dragon',{deathFunc:q.event.killDragon}); Create a simplified loot (aka chest): m.loot.simple('name',viewedIf,open); Ex: m.loot('Jewel Chest',q.event.gotJewel,q.event.giveJewel); Create a simplified loot (aka chest): m.toggle.simple('name',viewedIf,on,off); Ex: m.toggle('Door Switch',q.event.doorIsLocked,q.event.unlockDoor,q.event.lockDoor); ##### TEMPLATE to put in your quest file. q.minimizedMap = function(){ //create these actors when creating/loading the map m.actor.simple('Dragon',{deathFunc:q.event.killDragon}); m.loot.simple('Jewel Chest',q.event.gotJewel,q.event.giveJewel); m.toggle.simple('Door Switch',q.event.doorIsLocked,q.event.unlockDoor,q.event.lockDoor); }
