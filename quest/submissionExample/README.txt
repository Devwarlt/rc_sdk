####################################
A quest submission must include everything needed in order for the quest to work.

This can include:
-Quest Events
-Map (Image, Grid)
-Dialogue
-Items

Submission must include:
1- quest code (.js, file name is [yourQuestId].js) 
		This file contains info about dialogue, quest variables, maps, items etc...
2- map images in mapImage folder (put each map are in a separate folder)
3- tiled projects for the maps in tiledProject folder (put each project are in a separate folder)
	Only put the .json there. REMOVE the images as they are already in the mapImage folder.

4- info.js (general information ex: author/date/comments)
