### PREPARATION ### In order to test quests, you will need to run the game locally, you need to install the server on your computer. It's easy, don't worry. Video Link: STEPS: 1- Install Nodejs via "install Nodejs.exe" 2- Download the current version of the game on https://github.com/RainingChain/rainingchain (Click the Download ZIP at bottom right) 3- Unzip. 4- Click on STARTSERVER.bat to start the server. 8- Open a browser and go to http://localhost:3000/ Note: You will connect to the public online database automatically.
