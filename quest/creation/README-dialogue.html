#################################### DIALOGUE: Definitions: Npc: Who is talking. Ex: bob Face: Face appearing in the Chat Box when NPC talking (referring to Db.dialogue.face). Convo: What is the topic. Ex: Giving Quest Node: Each line of speech. Ex: Ask the player if he wants to help him Option: List of choice given to the player for a specific node. Selecting an option will change the current node. Options can also trigger functions. ################# General Template: q.dialogue['npc'] = {'face':'faceId', 'convo0':{ 'node0':{ ... }, 'node1':{ ... }, 'node2':{ ... }, }, 'convo1':{ 'node0':{ ... } } }; ################# Node Template: ## 1: Only text (no option): 'node0':{ 'text':"Hello", }, ## 2: Text with options (basic) 'node1':{ 'text':"Hello, how are you?", 'option':[ {'text':"Great.", 'next':{'node':'nextNode'}, }, {'text':"Bad.", 'next':{'node':'otherNextNode'} }, ] }, nextNode refers to the next node in the dialogue if the player chooses that option. It assumes you will stay in the same convo and the same npc. If you don't want that: {'text':"Bad.", 'next':{'npc':'nextNpc,'convo':'nextConvo','node':'nextNode'} }, ## 3: Text with options with functions 'node0':{ 'text':"Hello, how are you?", 'option':[ {'text':"Great.", 'next':{'node':'nextNode'}, 'func':(function(key){ ...functions... Ex: give item, give exp, change quest variable, teleport player }), 'param':[], }, {'text':"Bad.", ... }, ] }, When the player chooses an option, the game engine will trigger the function (if it has one). Advanced: The first function parameter will always be the player key. Second will be the quest variable for that player. Other parameters will be the ones you have put in the param array. #################################### COMMON DIALOGUE FUNCTIONS: Add Item: Itemlist.add(key,'itemId',amount); Remove Item: Itemlist.remove(key,'itemId',amount); Test if the player has item (return boolean): Itemlist.have(key,'itemId',amount); Give Exp: Skill.addExp(key,'skill',amount); Complete Quest: Quest.complete(key,'questId'); Change a Quest Variable: List.main[key].quest['questId']['questVariable'] = value; Teleport a player: Actor.teleport(key,x,y,'mapId'); Start a dialogue: Dialogue.start(key,{'group':'questId','npc':'bob','convo':'giveQuest','node':'intro0}); Add text in Chat Box: Chat.add(key,'text'); ### Examples: ### function(key){ if(List.main[key].quest['QmyQuest']['killedDragon'] === false){ Chat.add(key,'You haven't killed the dragon yet'); Actor.teleport(key,100,100,'dragonLair'); Itemlist.add(key,'fireEquipment',1); } if(List.main[key].quest['QmyQuest']['killedDragon'] === true){ Chat.add(key,'You have killed the dragon!'); Itemlist.remove(key,'fireEquipment',1); Skill.addExp(key,'magic',100); Quest.complete(key,'QmyQuest'); } }
