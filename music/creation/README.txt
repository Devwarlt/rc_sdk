### Creating song/sfx ####

Submission must include:
1- audio file (.mp3 for songs, .ogg/.mp3 for sfx)
2- info.js (general information ex: author/date)

Compression Settings for songs:

Mode: Variable
Rate (kbps): 110-150
Modus: Mono
Sample frequency: 22050 Hz

This should compress the audio file by 70-85%.

Note: Sfx do not need to be compressed.
