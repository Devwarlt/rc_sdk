### Creating icon ####

Submission must include:
1- image(s) (.png format)
2- info.js (general information ex: author/date)

2- info.js
Create a file named "info.js". 

Template
{
	"credits":["yourName","otherGuy"],	
	"date":"2014-03-10",
	"modifications":[],
	"comments":"The animation would look better in red."
}
End Template

credits: list of people who contributed to the submission
date: YYYY-MM-DD
modifications: list of modifications
comments: comments
